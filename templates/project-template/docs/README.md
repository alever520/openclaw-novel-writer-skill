# 参考资料目录

本目录用于存放创作参考资料。

---

## 用途

将任何有助于创作的资料放入此目录：
- 网文写作指南
- 爽文节奏分析
- 人物塑造技巧
- 世界观构建方法
- 其他优秀小说的分析
- ...

---

## 更新最佳实践

当你添加新的参考资料后，运行：

```
更新最佳实践
```

Novel Writer Skill 会自动：
1. 扫描本目录的新增内容
2. 提取关键要点
3. 更新 `references/writing-guide.md`
4. 应用到后续创作中

---

## 推荐资料类型

### 创作技巧
- 语言风格指南
- 节奏控制方法
- 对话写作技巧
- 战斗描写指南

### 分析文章
- 爆款小说分析
- 读者心理研究
- 类型小说规律总结

### 世界观素材
- 历史资料
- 科学知识
- 神话传说
- 文化背景

---

## 文件格式

推荐使用 Markdown 格式（`.md`），便于 Skill 解析。

如果是其他格式（PDF、Word、TXT），可以手动提取关键内容到 Markdown。

---
