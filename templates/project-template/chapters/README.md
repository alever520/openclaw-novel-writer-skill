# 章节目录

本目录用于存放小说章节。

---

## 命名规范

```
第XXX章-标题.md
```

示例：
- `第001章-新生.md`
- `第002章-废弃实验室.md`
- `第023章-浮空城升空.md`

---

## 章节模板

```markdown
# 第X章 标题

[章节正文]

---

(第X章完)
```

---

## 创作提示

1. 每章 3000-5000 字
2. 一章一个核心事件
3. 开头要承接上章
4. 结尾要留悬念
5. 完成后立即提交到 Git

---

## 使用 Novel Writer Skill 创作

```
写第X章
```

Skill 会自动：
- 根据大纲创作
- 进行自检
- 保存到本目录
- 提交到 Git
